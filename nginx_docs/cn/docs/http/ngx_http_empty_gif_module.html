配置范例
指令
     empty_gif

模块 ngx_http_empty_gif_module 只返回一个透明像素的GIF图片。

配置范例

location = /_.gif {
    empty_gif;
}

指令

语法: empty_gif;
默认值: —
上下文: location

在当前location里使用这个模块处理请求。

翻译: yzprofile
